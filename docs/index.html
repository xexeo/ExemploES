Descrição Informal

O Programa de Engenharia de Sistemas e Computação da COPPE/UFRJ realiza três seleções anuais para os cursos de Mestrado e Doutorado. Cada seleção envolve um cadastro que exige vários documentos e informações, uma análise prévia feita pela secretaria, e diversas avaliações (prova de currículo, prova escrita, prova oral, plano de projeto) onde apenas a prova de currículo é obrigatória e eliminatória, que envolvem a utilização do sistema pelos professores. No final, a secretaria volta a usar para definir o resultado final. Além disso há a possibilidade de recursos em cada prova, e também da comunicação entre candidatos, professores e secretaria pelo sistema, mas onde tudo é copiado via e-mails.

O sistema deve funcionar via web, mas de forma que seja possível usar em celulares (pelos candidatos). O sistema também deve permitir análises um a um dos candidatos, como análises comparativas (algumas listagens com informações de vários candidatos).
